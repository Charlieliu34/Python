一、Request-Reply

1.登入

a.登入指令

{
  "Request":"LOGIN",
  "Param":
  {
    "SystemName":"xxxxx",
    "ServiceKey":"xxxxx"
  }
}


b.登入回复

  • 登入验证成功
{
    "Reply":"LOGIN",
  "Success":"OK",
    "SessionKey":"XXXXXXXXX",
  "SubPort":"n"
}

注 1: SessionKey，之后所有 Request 指令一律需要带 SessionKey。SessionKey 错误，指令不执行。

注 2: SubPort 提供 client 用来建立第二条 ZeroMQ Pub/Sub 联机，用来接收主推回报。


  • 登入验证失败
{
    "Reply":"LOGIN",
  "Success":"Fail",
    "SessionKey":""
"ErrMsg":""
}


c.登出指令

{
 "Request":"LOGOUT",
 "SessionKey":"XXXXXXXXX"
}

6.期货/期权下单

a.下单指令

{
  "Request":"NEWORDER",
  "SessionKey":"XXXXXXXXX",
  "Param":
  {    
    "BrokerID":"xxxxx",
    "Account":"xxxxx",
    "Symbol":"TC.F.xxx.xxx.201710",
    "Price":"xxxxx",
    "StopPrie":"xxxxx",
    "Side":"n",
    ...
  }
}


b.下单回复

{
    "Reply":"NEWORDER",
  "Success":"OK"
}

注1: 代码一律采用TC代码。

注2: TimeInForce、OrderType、PositionEffect 列出所有可下单形式，请确认上手可支持下单形式，下单时代上正确的type。值接送上手Synthetic="0"，若要TCore洗价Synthetic="1"。



  BrokerID
  
BSTR
  
  券商
  

  Account
  
BSTR
  
  帳號
  

  Symbol
  
BSTR
  
  商品ID (包含復式商品ID)
  TC.F2.TWF.FITX.201604^FITX.201606
  

  Price
  
BSTR
  
  委託價格
  or
  欄位 + or - 價格
  欄位為固定字, 可為LAST or BID or
  ASK or MID or FILLED
  FILLED為OTO, OTOCO使用, 其他單不可使用FILLED
  ex. FILLED-15
  

  StopPrice
  
BSTR
  
  停損價
  or
  欄位 + or - 價格
  欄位為固定字, 可為LAST or BID or
  ASK or MID or FILLED
  FILLED為OTO, OTOCO使用, 其他單不可使用FILLED
  

  Side
  
BSTR
  
  買賣別 (複式單為整體買賣方向) 1:Buy 2:Sell
  

  TimeInForce
  
BSTR
  
  0 : None
  1 : ROD Day order
  2 : IOC | FAK Immediate or
  Cancel | Fill and Kill
  3 : FOK Fill or Kill
  4 : GTC Good-Till-Cancel
  5 : GTD Good-Till-Date
  6 : OPG 盤前預約單
  

  OrderType
  
BSTR
  
  0 : None Unknown
  1 : Market order (MARKET)
  2 : Limit order (LIMIT)
  3 : Stop order (STOP)
  4 : Stop limit order (STPLMT)
  5 : Trailing Stop (TSTOP)
  6 : Trailing StopLimit (TSTPLMT)
  7 : Market if Touched Order (MIT)
  8 : Limit if Touched Order (LIT)
  9 : Trailing Limit (TLMT)
  10 : 對方價(HIT)
  11 : 本方價(JOIN)
  12 : DOM-Triggered Stop ( DTS )
  13 : DOM-Triggered Stop Limit ( DTSL )
  14 : Breakeven (BE )
  15: 中間價(MID)
  
  
  20 : 最優價 (BST)
  21 : 最優價轉限價 (BSTL)
  22 : 五檔市價 (5LvlMKT)
  23 : 五檔市價轉限價 (5LvlMTL)
  24 : 市價轉限價 (MTL)
  25: 一定範圍市價(MWP)
  

  ContingentSymbol
  
BSTR
  
  追蹤Symbol
  僅Synthetic
  = 1
  OrderType=7 或 OrderType=8使用
  

  TrailingField
  
BSTR
  
  僅Synthetic
  = 1
  OrderType=5 或 OrderType=6使用
  0 : None
  1 : Last
  2 : Bid
  3 : Ask
  

  TrailingType
  
BSTR
  
  僅Synthetic
  = 1
  OrderType=5 或 OrderType=6使用
  0 : None
  1 : 依價格
  2 : 依TrailingPrice百分比
  3
  : 依TickSize幾跳
  

  TrailingAmount
  
BSTR
  
  僅Synthetic
  = 1
  OrderType=5 或 OrderType=6使用
  TrailingType=1時, TrailingAmount為百分比
  TrailingType=2時, TrailingAmount為點數
  TrailingType=3時, TrailingAmount為幾跳
  

  TouchPrice
  
BSTR
  
  僅Synthetic
  = 1
  OrderType=7 或 OrderType=8使用
  觸發價
  or
  欄位 + or - 價格
  欄位為固定字, 可為LAST or BID or
  ASK or FILLED
  FILLED為OTO, OTOCO使用, 其他單不可使用FILLED
  

  TouchField
  
BSTR
  
  僅Synthetic
  = 1
  OrderType=3 或 OrderType=4 或 OrderType=7 或 OrderType=8或OrderType=12 或 OrderType=13使用
  0 : None
  1 : Last
  2 : Bid
  3 : Ask
  

  TouchCondition
  
BSTR
  
  僅Synthetic
  = 1
  OrderType=7 或 OrderType=8使用
  0 : touch or 穿價
  1 : Greater
  2 : GreaterEqual
  3 : Equl
  4 : LessEqual
  5 : Less
  

  OrderQty
  
BSTR
  
  下單口數
  

  PositionEffect
  
BSTR
  
  0 : Open Open position
  1 : Close Close position
  2 : 平今
  3 : 平昨
  4 : Auto Auto select
  Open/Cloe position
  5 : 特殊自動單 有昨倉先平昨
  其它一律轉開倉
  10 : 備兌開倉
  11: 備兌平倉
  

  DayTrade
  
BSTR
  
  0 : None
  1 : 當沖
  

  Synthetic
  
BSTR
  
  0 : None
  1 : Synthetic
  

  GroupType
  
BSTR
  
  0 : None
  1 : Normal
  2 : OCO
  3 : OTO
  4 : OTOCO
  5 : OTOs(單線)
  

  GroupID
  
BSTR
  
  GroupID or OCOID or OTOID or OTOCOID or OTOSID
  

  ChasePrice
  
BSTR
  
  追價
  格式為[每次增加價格|追價幾次|每次追價幾秒|最後下市價或刪單或掛單(M or C or
  L)|PriceType]
  
  限價追價
  格式為[0|追價限制幾跳|-11 (Chase
  Trailing) or -12 (Chase if touch)|最後下市價或刪單或掛單(M or C or
  L)|PriceType]
  
  或是[1|追價幾次|-11 (Chase Trailing)
  or -12 (Chase if touch)|最後下市價或刪單或掛單(M or C or
  L)|PriceType]
  
  a.若 每次增加價格 < 1000 表示幾跳Ticks
  b.追價幾秒=-11 (Chase
  Trailing)
  c.追價幾秒=-12 (Chase if
  touch)
  d.Price Type=(LTP, BID, ASK)
  

  SlicedType
  
BSTR
  
  不支援群組單
  0 : None 5 :
  AllSliced After PriceTrigger
  1 : Iceberg
  2 : TimeSliced
  3 : VolumeSliced
  4 : AllSliced
  

  DiscloseQty
  
BSTR
  
  僅SlicedType =1 或 SlicedType =2 或SlicedType =3 或SlicedType =4或SlicedType =5使用
  逐筆揭露數量
  

  Variance
  
BSTR
  
  僅SlicedType =1 或 SlicedType =2或 SlicedType =3使用
  逐筆變動比例%
  

  Interval
  
BSTR
  
  僅SlicedType =2或 SlicedType =3使用
  SlicedType =2 : 間隔時間，ms
  SlicedType =3 : 間隔成交量
  

  LeftoverAction
  
BSTR
  
  僅SlicedType =2或 SlicedType =3使用
  0 : Leave 繼續掛著(預設)
  1 : Merge 取消該筆，剩餘數量加入下筆下單
  2 : Market 轉市價單
  3 : Payup 追價(未實做)
  

  SlicedPriceField
  
BSTR
  
  0 -> Fixed price mode
  非0 -> Relative
  price mode
  
  0 : None
  1 : Last
  2 : Bid
  3 : Ask
  4 : 對方價
  5 : 本方價
  

  SlicedTicks
  
BSTR
  
  SlicedPriceField
  != 0
  TickSize幾跳
  

  Breakeven
  
BSTR
  
  僅OrderType=14使用
  Breakeven觸發價
  or
  欄位 + or - 價格
  欄位為固定字, 可為LAST or BID or
  ASK or FILLED
  FILLED為OTO, OTOCO使用, 其他單不可使用FILLED
  

  BreakevenOffset
  
BSTR
  
  僅OrderType=14使用
  Breakeven停損價
  or
  欄位 + or - 價格
  欄位為固定字, 可為LAST or BID or
  ASK or FILLED
  FILLED為OTO, OTOCO使用, 其他單不可使用FILLED
  

  Threshold
  
BSTR
  
  僅OrderType=12或OrderType=13使用
  

  ExtCommands
  
BSTR
  
  延伸下單參數 多個參數請用；隔開
  1.自動單反向延遲 請帶
   DelayTransPosition=3000
  
  表延遲3秒
  2.下單前 將平倉掛單都刪單
   CancelCloseWorking=1 or 2
  3.下單速度是否調慢與設定相同
   FitOrderFreq=1
  

  Consecutive
  
BSTR
  
  1 : 連續送單 2:TW連續IOC
  

  NumberOfRetries
  
BSTR
  
  僅Consecutive=1使用
  持續送單幾次,
  NumberOfRetries>=9999 表示無線次數
  

  RetryInterval
  
BSTR
  
  僅Consecutive=1使用
  每次送單間隔ms,
  RetryInterval=0表是立即送單
  

  Strategy
  
BSTR
  
  策略名稱
  

  UserKey1
  
BSTR
  
  將可在回報的UserKey1欄位得到相同的字串
  



  下單錯誤碼
  

  -10
  
  Unknow Error
  

  -11
  
  買賣別不對
  

  -12
  
  複式單商品代碼解晰錯誤
  

  -13
  
  下單帳號, 不可下此交易所商品
  

  -14
  
  下單錯誤, 不支援的價格 或 OrderType 或 TimeInForce
  

  -15
  
  不支援證券下單
  

  -20
  
  連線未建立
  

  -22
  
  價格的TickSize錯誤
  

  -23
  
  下單數量超過該商品的上下限
  

  -24
  
  下單數量錯誤
  

  -25
  
  價格不能小於和等於0 (市價類型不會去檢查)
